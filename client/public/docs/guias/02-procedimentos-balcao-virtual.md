# Guias Jurídicos Completos para Limpeza de Nome - Procedimentos Práticos no Balcão Virtual

> Research Date: 2026-03-03

## Overview

Este documento apresenta um guia detalhado sobre os procedimentos práticos para utilização do Balcão Virtual do Tribunal de Justiça de São Paulo (tribunal) no contexto de limpeza de nome e negociação de dívidas. O Balcão Virtual é uma plataforma digital desenvolvida pelo tribunal que permite atendimento remoto via videoconferência, funcionando como um balcão físico virtual.

Este guia é especialmente relevante para o projeto CPF Blindado, pois fornece informações fundamentais sobre como os usuários podem utilizar os serviços virtuais do tribunal para obter orientações processuais, informações sobre processos de negativação, e participar de sessões de conciliação para acordos de dívidas.

É importante distinguir que o Balcão Virtual tribunal não é a mesma coisa que o sistema sistema judicial de peticionamento eletrônico. O Balcão Virtual é focado em atendimento ao vivo para orientações e informações, enquanto o sistema judicial é utilizado para protocolar ações judiciais.

## O que é o Balcão Virtual tribunal

O Balcão Virtual do tribunal é um serviço de atendimento remoto que permite videoconferência com unidades judiciais, oferecendo orientações processuais sem a necessidade de comparecimento presencial. O sistema foi implementado com base na Resolução CNJ nº 372/2021 e na Lei 11.419/2006, que dispõe sobre o uso de meios eletrônicos na tramitação de processos judiciais.

### Principais Características

- **Atendimento por videoconferência**: Utiliza a plataforma Microsoft Teams (não é necessário download, acesso via navegador)
- **Agendamento prévio**: A maioria dos serviços exige agendamento anterior
- **Horário de funcionamento**: Segunda a sexta-feira, das 9h às 17h (exceto feriados do tribunal)
- **Acesso gratuito**: Não há custos para utilizar o serviço
- **Atendimento por servidores treinados**: Servidores do próprio tribunal prestam as orientações

### Serviços Disponíveis

O Balcão Virtual oferece diversos serviços, incluindo:

- Orientações processuais sobre ações em andamento
- Informações sobre andamento de processos
- Esclarecimentos sobre prazos e procedimentos judiciais
- Suporte para questões relacionadas ao sistema eletrônico de processos
- Alguns serviços cartorários específicos
- Informações sobre competências para ajuizamento de ações

### Limitações Importantes

O Balcão Virtual possui algumas limitações que devem ser consideradas:

- **Apenas informações sobre processos em andamento**: O sistema não fornece orientações para casos que ainda não foram judicializados
- **Não substitui advogado**: As orientações são genéricas e não substituem a assistência jurídica profissional
- **Não realiza conciliações diretamente**: Para sessões de conciliação, é necessário verificar disponibilidade específica
- **Não protocola ações**: Para ajuizar ações, deve-se utilizar o sistema sistema judicial ou sistema judicial

## Acesso e Registro no Sistema

### Requisitos Técnicos

Para utilizar o Balcão Virtual tribunal, são necessários:

- Conexão estável com a internet
- Computador, tablet ou smartphone com câmera e microfone funcionais
- Navegador web atualizado (Chrome, Firefox, Edge)
- Documentos digitalizados em formato PDF
- Ambiente adequado para videoconferência (local silencioso e iluminado)

### Cadastro no Sistema

O acesso ao Balcão Virtual exige cadastro prévio, com procedimentos diferentes para advogados e partes não representadas:

#### Para Advogados

Advogados devem utilizar suas credenciais da OAB (Ordem dos Advogados do Brasil) para acesso ao sistema. O processo inclui:

1. Acessar o portal do Balcão Virtual através do site do tribunal
2. Selecionar a opção de acesso para advogados
3. Inserir número da OAB e senha cadastrada
4. Validar a identificação através do sistema

#### Para Partes não Representadas

Cidadãos sem representação advocatícia podem criar seu próprio cadastro:

1. Acessar o site oficial do Balcão Virtual: www.tjsp.jus.br/BalcaoVirtual
2. Selecionar a opção de cadastro para partes não representadas
3. Preencher formulário com dados pessoais completos:
   - Nome completo
   - Número do RG
   - Número do CPF
   - Endereço completo
   - Telefone para contato
   - E-mail válido
4. Validar os dados através do sistema de verificação do tribunal
5. Criar senha de acesso

### Verificação de Duas Etapas

A partir de 24/03/2025, o acesso ao portal sistema judicial (que integra alguns serviços do Balcão Virtual) passou a exigir código de verificação enviado para os e-mails cadastrados no sistema. Esta medida de segurança deve ser considerada ao acessar os serviços.

## Procedimento Passo a Passo para Utilização do Balcão Virtual

### Passo 1: Preparação Antecipada

Antes de agendar ou acessar o Balcão Virtual, prepare-se adequadamente:

**Documentação Necessária**

- Documento de identidade oficial (RG) com foto
- CPF (documento ou número)
- Comprovante de residência atualizado
- Todos os documentos relacionados ao processo ou dívida:
  - Contratos originais
  - Extratos de débito
  - Comprovantes de pagamento já realizados
  - Notificações recebidas
  - Certidões de órgãos de proteção ao crédito (bureau de credito, servico de protecao ao credito, etc.)
- Número do processo (se já houver ação judicial em andamento)
- Dados completos da parte contrária (credor ou devedor)

**Preparação do Ambiente**

- Teste câmera e microfone com antecedência
- Verifique a estabilidade da conexão com a internet
- Escolha um local silencioso e adequado para a conversa
- Tenha papel e caneta para anotações
- Prepare uma lista de perguntas e esclarecimentos necessários

### Passo 2: Acesso ao Sistema

1. Acesse o site oficial: www.tjsp.jus.br/BalcaoVirtual
2. Faça login com suas credenciais (OAB para advogados, cadastro pessoal para cidadãos)
3. Aguarde o código de verificação por e-mail (se aplicável)
4. Insira o código de verificação no sistema

### Passo 3: Seleção do Serviço

O sistema apresentará as opções de serviço disponíveis. Para questões relacionadas à limpeza de nome e dívidas, as opções mais relevantes são:

- **Orientações Processuais**: Para tirar dúvidas sobre procedimentos judiciais
- **Informações sobre Processos**: Para consultar andamento de ações já ajuizadas
- **Serviços de Conciliação**: Se houver sessão de conciliação agendada

Selecione o serviço adequado à sua necessidade. É fundamental escolher a categoria correta para ser encaminhado ao setor competente.

### Passo 4: Agendamento (quando necessário)

Alguns serviços exigem agendamento prévio. O procedimento inclui:

1. Escolher o tipo de serviço desejado
2. Selecionar data e horário disponível
3. Confirmar o agendamento
4. Anotar o número de protocolo ou confirmação

**Atenção**: A pontualidade é essencial. Atrasos podem resultar no cancelamento do atendimento.

### Passo 5: Participação da Videoconferência

No horário agendado:

1. Acesse novamente o Balcão Virtual
2. Clique na opção de entrada na sala de atendimento virtual
3. Insira seu nome completo na página do Microsoft Teams que será aberta
4. Aguarde o servidor iniciar o atendimento
5. Durante a sessão:
   - Mantenha a câmera ligada (preferencialmente)
   - Fale de forma clara e objetiva
   - Tenha os documentos à mão para apresentação quando solicitado
   - Tire notas das orientações fornecidas
   - Faça perguntas claras e específicas

### Passo 6: Encerramento e Avaliação

Ao final do atendimento:

1. Confirme todas as orientações recebidas
2. Anote números de processo, protocolos ou informações importantes
3. Preencha a avaliação do serviço (geralmente solicitada ao final)
4. Salve ou imprima o comprovante de atendimento, se disponível

## Roteiro de Fala para Atendimento no Balcão Virtual

### Abertura do Atendimento

Ao iniciar a conversa com o servidor, utilize uma abordagem clara e objetiva:

```
"Bom dia/tarde. Meu nome é [seu nome completo], CPF [número do CPF].
Estou buscando orientações sobre [descrever brevemente o assunto:
uma dívida negativada / um processo em andamento / como proceder
para limpeza do meu nome]."
```

### Descrição do Caso

Apresente seu caso de forma organizada:

**Informações Essenciais**

1. **Contexto da Dívida**:
   - "A dívida originated-se de [origem: contrato de empréstimo / compra parcelada / serviços não prestados]"
   - "O valor original era de [valor]"
   - "A dívida está negativada desde de [data]"
   - "Já realizei pagamentos no valor de [valor], em [datas]"

2. **Situação Atual**:
   - "Atualmente [descrever situação: estou desempregado / tenho renda de X / posso pagar parcelas de Y]"
   - "Tenho interesse em [descrever objetivo: negociar / limpar o nome / verificar processos]"

3. **Dúvidas Específicas**:
   - "Gostaria de saber [pergunta específica: quais são meus direitos / como funciona a conciliação / quais documentos necessários]"

### Perguntas Recomendadas

Durante o atendimento, faça perguntas objetivas para obter as informações necessárias:

**Sobre Procedimentos**
- "Quais são os procedimentos adequados para o meu caso?"
- "É necessário ajuizar uma ação ou existe possibilidade de acordo extrajudicial?"
- "Quais são os prazos legais aplicáveis?"
- "Quais documentos adicionais são necessários?"

**Sobre Conciliação**
- "Existe sessão de conciliação agendada para o meu processo?"
- "Como posso solicitar uma conciliação?"
- "Quais são as possibilidades de acordo para este tipo de dívida?"
- "O acordo pode incluir descontos e parcelamento?"

**Sobre Limpeza de Nome**
- "Após a quitação/acordo, em quanto tempo meu nome é limpo dos órgãos de proteção ao crédito?"
- "É necessário providenciar a baixa da negativação ou ela é automática?"
- "Como posso comprovar a limpeza do meu nome para futuras consultas?"

### Durante a Negociação (se aplicável)

Caso o atendimento envolva uma sessão de conciliação ou negociação:

**Postura Adequada**
- Mantenha a calma e o respeito profissional
- Seja honesto sobre sua capacidade financeira
- Demonstre genuíno interesse em resolver a situação
- Não faça promessas de pagamento que não possa cumprir

**Argumentação para Descontos**
- "Considerando o tempo decorrido desde a contratação..."
- "Diante da minha atual situação financeira..."
- "Levando em conta que o valor original era significativamente menor..."
- "Estou disposto a quitar à vista, se houver desconto significativo..."

**Propostas de Parcelamento**
- "Posso comprometer [X]% da minha renda mensal"
- "Consigo pagar parcelas de [valor] em [número] vezes"
- "A primeira parcela pode ser maior, com redução nas subsequentes"

### Encerramento do Atendimento

Antes de finalizar:

```
"Agradeço pelas orientações. Para confirmar: [liste os pontos principais:
devo protocolar ação X / o próximo passo é Y / o prazo é Z].
Existe mais alguma providência que eu deva tomar agora?"
```

## Conciliação pelo Balcão Virtual

### O que é Conciliação Judicial

A conciliação judicial é um método alternativo de resolução de conflitos no qual um terceiro (conciliador ou juiz) facilita o diálogo entre as partes para encontrar um acordo mutuamente aceitável. No contexto de dívidas negativadas, a conciliação pode resultar em:

- Descontos sobre o valor total da dívida
- Parcelamento do débito em condições favoráveis
- Limpeza do nome como condição do acordo
- Suspensão de ações judiciais em andamento

### Diferença entre Conciliação, Mediação e Arbitragem

| Método | Características | Papel do Terceiro | Decisão Final |
|--------|----------------|-------------------|---------------|
| **Conciliação** | Mais informal, focada em acordo | Conciliador sugere soluções | Partes decidem |
| **Mediação** | Processo mais longo, focado em relacionamento | Mediador facilita diálogo | Partes decidem |
| **Arbitragem** | Mais formal, similar a processo | Árbitro analisa provas | Árbitro decide (obrigatório) |

### Procedimento de Conciliação no tribunal

Embora o Balcão Virtual possa fornecer informações sobre conciliação, as sessões de conciliação propriamente ditas geralmente ocorrem através de agendamento específico nos Centros Judiciários de Conciliação (Cejuscs).

**Passos para Solicitação de Conciliação**

1. **Verificação da Disponibilidade**
   - Consulte se o seu caso é passível de conciliação
   - Verifique se já existe audiência de conciliação designada no processo

2. **Protocolo de Pedido**
   - Alguns casos permitem pedido de conciliação através do sistema sistema judicial
   - Outros requerem petição específica ao juízo competente
   - O Balcão Virtual pode orientar sobre o procedimento adequado

3. **Preparação para a Sessão**
   - Reúna toda a documentação relevante
   - Prepare uma proposta de acordo realista
   - Calcule sua capacidade de pagamento
   - Defina seus limites mínimos e máximos de negociação

### Roteiro para Sessão de Conciliação Virtual

**Abertura da Sessão**

```
"Senhor(a) Conciliador(a), meu nome é [nome], representante [próprio / de X].
Estou aqui para buscar uma solução para o processo [número],
referente à dívida de [valor] com [credor]."
```

**Apresentação de Propostas**

```
"Após analisar minha situação financeira, posso propor:
1. Pagamento à vista de [valor] com desconto de [X]%
OU
2. Parcelamento em [número] vezes de [valor], com primeira parcela em [data]"
```

**Durante as Negociações**

- Mantenha postura colaborativa
- Esteja preparado para fazer concessões razoáveis
- Anote todos os termos discutidos
- Solicite esclarecimentos sobre pontos não compreendidos

**Formalização do Acordo**

Se houver acordo:

```
"Só para confirmar os termos do acordo:
- Valor total acordado: [valor]
- Forma de pagamento: [à vista / parcelado]
- Número de parcelas: [número]
- Valor de cada parcela: [valor]
- Data da primeira parcela: [data]
- Compromisso de baixa da negativação em: [prazo]"
```

**Se Não Houver Acordo**

Caso a conciliação não resulte em acordo:

```
"Entendo que não foi possível达成 acordo neste momento.
Gostaria de saber quais são os próximos procedimentos processuais
e se há possibilidade de nova tentativa de conciliação no futuro."
```

## Documentação Necessária

### Documentos Pessoais

- **Documento de Identidade (RG)**: Original ou cópia autenticada, válido e com foto
- **CPF**: Documento ou número; para estrangeiros, documento de estrangeiro (RNE/CRNM)
- **Comprovante de Residência**: Atualizado (últimos 3 meses), em nome do interessado ou com declaração
- **Certidão de Casamento ou Nascimento**: Dependendo do tipo de ação

### Documentos da Dívida

- **Contrato Original**: Se houver contrato assinado
- **Extratos Atualizados**: Mostrando o valor atual da dívida
- **Histórico de Pagamentos**: Comprovantes de pagamentos já realizados
- **Notificações**: Cartas de cobrança, notificações extrajudiciais, intimações
- **Certidões de Proteção ao Crédito**: Atualizadas, de bureau de credito, servico de protecao ao credito, cadastro de protecao ao credito, cadastro de protecao ao credito

### Documentos Processuais (se aplicável)

- **Número do Processo**: Essencial para consultas
- **Petição Inicial**: Se a ação já foi ajuizada
- **Contestação**: Se já apresentada
- **Cópias de Decisões**: Sentenças, acórdãos ou decisões interlocutórias
- **Certidões Processuais**: Demonstrando andamento do processo

### Documentos Comprobatórios da Situação Financeira

Para negociação de dívidas, podem ser necessários:

- **Holerites ou Contracheques**: Últimos 3 meses
- **Declaração de Imposto de Renda**: Última declaração entregue
- **Extratos Bancários**: Demonstrando movimentação financeira
- **Comprovantes de Despesas**: Aluguel, contas, escolas, etc.
- **Declaração de Insuficiência**: Se aplicável, para benefícios da justiça gratuita

## Prazos e Prazos Legais Relevantes

### Prazos para Limpeza de Nome

**Após Quitação da Dívida**

- **Comunicação do Credor**: Até 5 dias úteis para comunicar a baixa aos órgãos de proteção ao crédito
- **Atualização dos Órgãos**: Geralmente de 3 a 7 dias úteis após receberem a comunicação
- **Total**: Em média 10 a 15 dias úteis após a quitação

**Negativação Indevida**

- **Aviso Prévio**: O credor deve enviar notificação antes de negativar (prazo mínimo: 10 dias antes)
- **Ausência de Aviso**: Se não houve notificação prévia, a negativação pode ser considerada ilegal
- **Liminar para Retirada**: Em casos de negativação indevida, é possível obter liminar para retirada imediata

### Prazos Processuais

**Prescrição de Dívidas**

- **Título Executivo Extrajudicial**: 5 anos (cheque, nota promissória, duplicata)
- **Dívidas Comuns**: 3 a 10 anos, dependendo do tipo
- **Débitos Bancários**: Geralmente 5 anos

**Prazos para Resposta em Processo**

- **Réplica à Contestação**: 15 dias
- **Recurso Inominado**: 10 dias (Juizado Especial)
- **Apelação**: 15 dias (Justiça Comum)

## Alternativas ao Balcão Virtual

Quando o Balcão Virtual não for adequado ao caso, existem outras opções:

### sistema judicial tribunal (Peticionamento Eletrônico)

Utilizado para protocolar ações judiciais:

1. Acesse: esaj.tjsp.jus.br
2. Cadastre-se ou faça login
3. Selecione "Peticionamento Eletrônico"
4. Escolha o tipo de ação (ex: "Negativação Indevida", "Consignação em Pagamento")
5. Preencha os dados e anexe os documentos
6. Envie a petição eletronicamente

### sistema judicial (Novo Sistema do tribunal)

O tribunal está em transição para o sistema sistema judicial:

- Verifique se a competência da ação está disponível no sistema judicial antes de protocolar
- Consulte o cronograma de migração no site do tribunal
- Alguns tipos de ação já estão sendo ajuizados diretamente no sistema judicial

### Atendimento Presencial

Para casos que exigem atendimento presencial:

- Consulte os endereços e horários no site do tribunal
- Verifique se é necessário agendamento prévio
- Leve toda a documentação original e cópias

### Outros Canais de Atendimento

- **WhatsApp tribunal**: Disponível para algumas consultas e orientações
- **Telefone de Atendimento**: Consulte os números disponíveis no site
- **Ouvidoria**: Para reclamações sobre serviços prestados
- **Defensoria Pública**: Para assistência jurídica gratuita a quem não pode pagar advogado

## Dicas Práticas e Recomendações

### Antes do Atendimento

1. **Organize toda a documentação**: Tenha tudo digitalizado e em PDF
2. **Prepare um resumo do caso**: Escreva os principais pontos para não esquecer
3. **Calcule sua capacidade de pagamento**: Saiba exatamente quanto pode oferecer
4. **Defina seus objetivos**: O que você considera um acordo satisfatório?
5. **Teste seu equipamento**: Câmera, microfone e conexão

### Durante o Atendimento

1. **Seja pontual**: Atrasos podem cancelar o atendimento
2. **Mantenha postura profissional**: Trate o servidor com respeito
3. **Seja claro e objetivo**: Vá direto ao ponto
4. **Tire notas**: Anote todas as orientações recebidas
5. **Faça perguntas**: Não saia com dúvidas

### Depois do Atendimento

1. **Revise suas anotações**: Confirme se tudo foi registrado corretamente
2. **Cumpra os prazos**: Siga todas as orientações nos prazos indicados
3. **Guarde comprovantes**: Protocolos, certidões, anotações
4. **Acompanhe os processos**: Use o sistema de consulta processual
5. **Busque assistência jurídica**: Se necessário, consulte um advogado

### Erros Comuns a Evitar

- **Faltar documentos importantes**: Verifique tudo antes do atendimento
- **Ser agressivo ou desrespeitoso**: Isso prejudica seu atendimento
- **Ocultar informações**: Seja honesto sobre sua situação
- **Fazer promessas irrealistas**: Ofereça apenas o que pode cumprir
- **Ignorar prazos**: Prazos processuais são rigorosos
- **Subestimar a importância da orientação profissional**: O Balcão Virtual não substitui advogado

## Considerações Especiais para Limpeza de Nome

### Verificação Prévia

Antes de buscar a limpeza do nome:

1. **Consulte todos os órgãos de proteção ao crédito**:
   - bureau de credito: bureau de credito.com.br
   - servico de protecao ao credito: servico de protecao ao creditobrasil.org.br
   - cadastro de protecao ao credito: boavistaservicos.com.br
   - cadastro de protecao ao credito: cadastro de protecao ao credito.com.br

2. **Anote todas as dívidas**:
   - Valor original
   - Valor atualizado
   - Credor
   - Data da negativação
   - Número do contrato (se houver)

3. **Verifique se há ações judiciais**:
   - Consulte processos em seu nome no site do tribunal
   - Verifique se existe execução judicial em andamento

### Estratégias de Negociação

**Para Dívidas Antigas**

- Verifique a prescrição (dívidas muito antigas podem estar prescritas)
- Negocie descontos maiores (credores têm interesse em receber algo a nada)
- Considere ofertas de quitação com desconto significativo

**Para Dívidas Recentes**

- Negocie parcelamento com parcelas compatíveis com sua renda
- Considere pagamento à vista se tiver desconto atrativo
- Verifique possibilidade de carência (pagamento da primeira parcela em 60-90 dias)

**Para Múltiplas Dívidas**

- Priorize dívidas com juros mais altos
- Negocie parcelamentos globais (se possível)
- Consolide dívidas se houver vantagem financeira

## Casos Especiais

### Negativação Indevida

Se você foi negativado injustamente:

1. **Reúna provas da indevidade**:
   - Comprovantes de pagamento
   - Erros na identificação
   - Dívida já quitada
   - Ausência de notificação prévia

2. **Consulte o Balcão Virtual** para orientações sobre:
   - Como proceder para retirada da negativação
   - Necessidade de ação judicial
   - Possibilidade de liminar para retirada imediata

3. **Considere ação judicial** para:
   - Retirada da negativação
   - Indenização por danos morais
   - Reembolso de custos

### Dívidas Prescritas

Dívidas prescritas não podem ser cobradas judicialmente, mas podem constar nos órgãos de proteção ao crédito:

1. **Verifique a prescrição**:
   - Conte o prazo a partir do vencimento
   - Considere interrupções (cobrança, reconhecimento da dívida)
   - Consulte advogado para análise precisa

2. **Exija a retirada**:
   - solicite a baixa ao credor
   - Proteste nos órgãos de proteção ao crédito
   - Aja judicialmente se necessário

### Dívidas de Valores Pequenos

Para dívidas de pequeno valor (até 20 salários mínimos):

- O Juizado Especial Cível é competente
- Procedimento mais rápido e informal
- Custas processuais reduzidas
- Possibilidade de conciliação obrigatória

## Key Takeaways

1. **O Balcão Virtual tribunal é uma ferramenta de orientação, não de resolução direta de conflitos**: Ele serve para tirar dúvidas e obter informações sobre processos, mas não substitui o ajuizamento de ações ou a participação em sessões de conciliação formal.

2. **Preparação é essencial**: Tenha todos os documentos organizados, suas perguntas preparadas e uma proposta realista de negociação antes de acessar o serviço.

3. **Pontualidade e respeito são fundamentais**: Atrasos podem cancelar o atendimento, e postura profissional facilita a obtenção das orientações necessárias.

4. **O Balcão Virtual não substitui advogado**: Para ações judiciais, estratégias complexas ou situações de alto risco, a assistência de um profissional qualificado é indispensável.

5. **Documente tudo**: Anote todas as orientações recebidas, guarde comprovantes de atendimento e mantenha cópias de todos os documentos apresentados.

6. **Verifique a prescrição**: Antes de pagar dívidas antigas, verifique se estão prescritas. Dívidas prescritas não podem ser cobradas judicialmente.

7. **Após quitação, acompanhe a baixa**: O credor tem 5 dias úteis para comunicar a baixa aos órgãos de proteção ao crédito. Se isso não ocorrer, tome providências.

8. **Considere alternativas extrajudiciais**: Plataformas como plataforma de negociacao de dividas oferecem negociação direta com descontos significativos, sem necessidade de processo judicial.

9. **Conheça seus direitos**: O Código de Defesa do Consumidor protege contra cobranças abusivas e exige notificação prévia antes da negativação.

10. **Mantenha registro de todos os acordos**: Qualquer acordo de dívida deve ser formalizado por escrito, com todas as condições claras e assinaturas de todas as partes envolvidas.
